# 근무기록 Excel 파일 구조 분석 리포트

**작성일**: 2026-03-03
**분석 대상**: 근무기록 공양식.xlsm / 근무기록 취합 데이터.xlsm
**분석 방법**: Python openpyxl (data_only=True) 직접 파싱

---

## 핵심 요약

두 파일은 **부서별 근무기록 입력(공양식) → 전사 취합(취합 데이터)** 구조로 연결된다.
공양식은 부서 담당자가 매월 계획 입력 → 기록 추가 → 마감 제출하는 운영 파일이고,
취합 데이터는 전체 부서 공양식의 계산 결과를 파워쿼리로 병합한 분석용 파일이다.

시간대별 인력 현황 계산에 필요한 핵심 컬럼은 **`출근시각`, `퇴근시각`, `복무결과`, `근무종류`** 4개다.

**역할**: 부서 담당자가 매월 초 직원별 근무 계획을 입력하는 시트.

## 4. 두 파일의 관계

```
[부서 담당자 운영]                    [취합/분석 담당자]
공양식.xlsm (부서별)                  취합 데이터.xlsm
  │                                      │
  ├─ [계획] 시트 입력                    ├─ [ref] 시트: 취합 원본 경로 설정
  ├─ [기록] 시트 추가                    │
  ├─ [계산] 자동 계산                    └─ 파워쿼리로 각 시트 취합
  └─ [마감] 시트 → 마감 완료 파일            ├─ ⓐ계산 ← 각 부서 [계산] 취합
                                            ├─ ⓑ기록 ← 각 부서 [기록] 취합
         취합 원본 파일                     └─ ⓒ계획 ← 각 부서 [계획] 취합
    ③근무기록 취합(2026-01)_마감.xlsm
                 ↑
    각 부서 공양식 마감 데이터를
    별도 취합 파일로 통합
```

**연결 키 구조**:

1. **마감시트주소**: `YYYYMMDD + 직원번호` 형태 (예: `2026030101` = 2026-03-01의 1번 직원)
2. **날짜사번**: `엑셀날짜시리얼 + 사번` 형태 (예: `46023S1705199`)
3. **계획위치**: 명단 시트의 위치값 → 계산 시트의 행 위치 매핑
4. **기록그룹**: 부서 단위 식별자 (예: "상담1부", "민원관리부")

**데이터 흐름**:
```
명단.xlsm (위치값) ─┐
                    ├→ 계산 시트 (자동 계산)
계획 시트 (계획코드) ─┤         ↓
                    │  마감시트주소로 연결
기록 시트 (보정 이벤트) ─┘
                                 ↓
                         마감 완료 파일
                                 ↓
                    파워쿼리 (취합 데이터)
                                 ↓
                    ⓐ계산 + ⓑ기록 + ⓒ계획
```

# 특정 시각 T에 근무 중인 직원 필터 조건
is_working = (
    (row['출근시각'] != '') and           # 출근 기록 있음
    (row['복무결과'] not in ['휴', '연차', '안휴', '육휴', '병가', '장휴', '0']) and  # 비출근 아님
    (row['출근시각'] <= T) and            # T 이전에 출근

### 6-3. 주의사항

- **출근시각 빈값 문제**: 총 12,090행 중 4,722건이 빈값 (39.1%). 공휴일/휴가 행이 포함됨. `복무결과 != '휴'` 조건 선행 필요.

- **출근시각과 퇴근시각 타입**: timedelta 형태로 실제 시각(예: 9:00:00)을 저장.

## 7. 데이터 품질 이슈 및 특이사항

### 7-1. 타입 혼재 문제 (심각도: 중)

**계획출근 컬럼**: 동일 컬럼에서 두 가지 타입이 혼재됨
- 빈 문자열 `""` : 비출근일 (4,720건)
- 소수 분수 `0.375` : 9:00 시각을 1일 기준 비율로 저장 (4,631건)

변환 공식: `시각(시) = 소수값 × 24` → `0.375 × 24 = 9.0시`

**날짜 컬럼 (ⓑ기록)**: 엑셀 날짜 시리얼 정수로 저장
- `46024` = 2026-01-02 (엑셀 기준 1900-01-00부터 카운트)

**직원 컬럼 (ⓑ기록)**: 직원 번호가 `datetime` 타입으로 잘못 저장됨
- `1900-01-05` = 직원번호 5번을 날짜로 해석한 것
